How to join 2 columns in a data set to make a separate column in SQL

I've tried searching online but most examples are too complex for my level. Can someone explain the simplest way to do this?

In SQL, you can use the CONCAT function to combine two columns:

SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM users;

This will create a new column called full_name that combines the first_name and last_name columns with a space in between.

Depending on your database system, you might also use the concatenation operator:

  • In MySQL: SELECT first_name || ' ' || last_name AS full_name FROM users;
  • In SQL Server: SELECT first_name + ' ' + last_name AS full_name FROM users;

But the CONCAT function is more universally supported.
